`enableTime` - Enables time picker (Включает выбор времени)

`time_24hr` - Displays time picker in 24 hour mode without AM/PM selection when
enabled.(Отображает средство выбора времени в 24-часовом режиме без выбора
AM/PM, если включено.)

`defaultDate` - Sets the initial selected date(s). (Устанавливает начальную
выбранную дату (даты).)  
you can supply a single Date object or a date string. (вы можете предоставить
один объект Date или строку даты.)

`minuteIncrement` -Adjusts the step for the minute input (incl. scrolling)
(Регулирует шаг ввода минут (включая прокрутку))

`onClose` - Function(s) to trigger on every time the calendar is closed  
(Функция(и), которая срабатывает каждый раз, когда календарь закрывается)

selectedDates – массив объектов Date, выбранных пользователем. Когда даты не
выбраны, массив пуст.
